PediDoctor

A saúde do seu filho nas palmas das suas mãos

Acessar sistema

Contextualização

Essa é Mariana, filha de Beatriz

  • Nascida em Belo Horizonte
  • Desde dois anos de idade enfrenta problemas com bronquite

Aos 4 anos, Beatriz e sua família se mudaram para Ipatinga

A partir desse momento, os médicos tiveram dificuldades nas consultas, pois todo o histórico médico de Mariana permaneceu no hospital da sua antiga cidade.

Solução: PediDoctor

Software que tem o objetivo de centralizar as informações médicas que são importantes para a vida da criança, além de permitir um diagnóstico compartilhado entre os diversos setores de saúde.




Método de Desenvolvimento

Definição do escopo

Verificação do processo básico de marcação e acompanhamento de consultas em clínicas médicas.

Também foram levantadas hipóteses sobre quais informações são importantes para os pais e os médicos

Ferramentas utilizadas

  • Backend: Servidor em Node.js, com banco de dados MongoDb e uso da tecnologia GraphQl.
  • Frontend: Framework Algular 2+, com Material Design
  • Versionamento: Ferramenta Git, com servidor GitLab e cliente SmartGit/GitKraken

Processo de Desenvolvimento

  • Cronograma baseado no Scrum: Foram realizadas iterações para entregas quinzenais, com priorização daquilo que mais agrega valor aos usuários.
  • Também foram adotadas algumas práticas do Extreme Programing, como: Integração contínua, padrões de codificação e projeto simples

Distribuição das atividades

Todos participaram da implementação de todas as funções, em front-end e backend, mas houve algumas distribuições:

  • André: Aspectos mais específicos do backend: Modelo de dados e API
  • Leonam: Aspectos mais específicos do frontend: Padrões de interface e Fluxos de usabilidade
